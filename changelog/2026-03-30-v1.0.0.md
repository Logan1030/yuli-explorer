# 于里探索家 v1.0.0

## 概述
于里探索家是一款面向4-10岁儿童的三科学习乐园App，包含英语城堡、数学迷宫和语文花园三个学习模块。

## 功能特性

### 科目模块
- **英语城堡** 🏰 - 5章词汇学习，包含基础词汇、动物世界、数字颜色、家庭成员、自然世界
- **数学迷宫** 🔢 - 10关数学题目，包含加法、减法、乘法、除法、大小比较、序数认知、简单应用
- **语文花园** ✏️ - 5章汉字学习，包含基础汉字、自然万物、数字与量词、动物象形、人体与动作

### 学习特性
- 星星奖励系统（每关最多3星）
- 关卡逐步解锁
- 即时反馈（正确/错误提示）
- Confetti庆祝动画
- TTS语音朗读（英语、语文）
- 学习进度持久化（Hive本地存储）

### 界面特色
- 卡片式设计
- 渐变色主题
- 底部导航栏
- 我的小窝星星总览

## 技术栈
- Flutter 3.41+
- Provider (状态管理)
- go_router (路由)
- Hive (持久化)
- flutter_tts (语音朗读)
- confetti (庆祝动画)
