# 于里探索家 v1.0.0

## 概述
于里探索家是一款面向4-10岁儿童的三科学习乐园App，包含英语城堡、数学迷宫和语文花园三个学习模块。

## 功能特性

### 科目模块
- **英语城堡** 🏰 - 5章词汇学习，包含基础词汇、动物世界、数字颜色、家庭成员、自然世界
- **数学迷宫** 🔢 - 10关数学题目，包含加法、减法、乘法、除法、大小比较、序数认知、简单应用
- **语文花园** ✏️ - 5章汉字学习 + 拼音花园

### 拼音花园 🔤
- **声母练习** - 23个声母（b p m f d t n l g k h j q x zh ch sh r z c s y w）
- **单韵母练习** - 6个单韵母（a o e i u ü）
- **声调练习** - 四声+轻声
- **整体认读** - 16个整体认读音节（zhi chi shi ri zi ci si yi wu yu ye yue yuan yin yun ying）

### 学习特性
- 星星奖励系统（每关最多3星）
- 关卡逐步解锁
- 即时反馈（正确/错误提示）
- Confetti庆祝动画
- TTS语音朗读（英语、语文、拼音）
- 学习进度持久化（Hive本地存储）

### 界面特色
- 卡片式设计
- 渐变色主题
- 底部导航栏
- 我的小窝星星总览

### 新增数据结构（chinese_data.dart）
```dart
// 拼音基础数据
initials              // 23个声母
simpleVowels          // 6个单韵母
compoundVowels        // 8个复韵母
nasalVowels          // 9个鼻韵母
medialVowels         // 3个介母
tones                // 声调（4声+轻声）
wholeSyllables       // 16个整体认读音节
syllableSeparators   // 隔音符号

// Quiz数据
PinyinQuizType       // 枚举：声母/韵母/声调/整体认读
PinyinQuestion       // 拼音题目
PinyinChapter        // 拼音章节
pinyinChapters       // 4个拼音章节
```
